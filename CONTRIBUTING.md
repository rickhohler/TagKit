# Contributing

Thank you for your interest in TagKit!

**Please Note:** At this time, `TagKit` is closed to external contributions. Use of this library is encouraged, but we are not currently accepting Pull Requests from the public.

If you encounter a bug or have a feature request, please [submit an issue](https://github.com/rickhohler/TagKit/issues/new/choose) using the provided templates.

## Reporting Bugs

Please use the **Bug Report** template to report any issues you find.

## Feature Requests

Please use the **Feature Request** template to suggest new features or improvements.
