## Reporting a Vulnerability

We take the security of TagKit seriously. If you have found a vulnerability, please report it to us immediately.

### How to Report

Please report vulnerability to [maintainer-email@example.com] or via private dismissal on GitHub.

**Do not report security vulnerabilities through public GitHub issues.**

We will acknowledge your report within 48 hours and will prioritize a fix.
